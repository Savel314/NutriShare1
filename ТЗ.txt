Создал Telegram-бота для учета рецептов, их обмена между пользователями и автоматического подсчета калорий. 
Бот написан на Python и обладает командами
2. Основные функциональные требования
2.1. Базовые команды
/start — Начало работы. Приветственное сообщение с кратким описанием возможностей бота.

/help — Справка. Вывод списка команд с пояснениями.

/stop_recipe — Прервать ввод рецепта (если пользователь начал создание).
2.2. Работа с рецептами
/add_recipe — Пошаговый ввод рецепта:

Название.

Ингредиенты (название продукта, вес/количество).

Шаги приготовления (поэтапно).

Автоматический расчет калорийности на основе введенных продуктов.

/show_all_recipes — Показать все рецепты в боте (с пагинацией, если их много).

/show_my_recipes — Показать рецепты, добавленные текущим пользователем.

/recipe <название_рецепта> — Поиск и вывод рецепта по названию.
2.3. Обмен рецептами
Возможность делиться рецептами с другими пользователями через:

Прямую отправку в чат.
2.4. Подсчет калорий
Автоматический расчет калорий для каждого рецепта на основе базы данных продуктов (название продукта → калории на 100 г).

3. Технические требования
Язык разработки: Python 3.9+.

Библиотеки:

python-telegram-bot для работы с Telegram API.
